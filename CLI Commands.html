Basic File and Directory Commands

  1. ls - List directory contents.
  2. cd - Change directory.
  3. pwd - Print working directory.
  4. cp - Copy files or directories (e.g., cp source.txt destination.txt).
  5. mv - Move or rename files or directories.
  6. rm - Remove files or directories (use -r for directories).
  7. mkdir - Create a new directory.
  8. rmdir - Remove an empty directory.
  9. touch - Create an empty file or update a file's timestamp.

File Viewing and Editing

  1. cat - Concatenate and display file content.
  2. less - View file contents one page at a time.
  3. nano - Simple text editor in the terminal.
  4. vim - Advanced text editor in the terminal.

System Commands

  1. top - Display running processes and system resource usage.
  2. ps - Display currently running processes.
  3. kill - Terminate a process by PID.
  4. df - Report file system disk space usage.
  5. du - Estimate file and directory space usage.

Networking

  1. ping - Test network connectivity to a host.
  2. ifconfig - Display network configuration.
  3. netstat - Display network connections and routing tables.
  4. curl - Transfer data from or to a server (e.g., download files).

File Permissions

  1. chmod - Change file permissions.
  2. chown - Change file ownership.

Miscellaneous

  1. man - Display the manual for a command.
  2. history - Show command history.
  3. clear - Clear the terminal screen.
  4. echo - Display a line of text.
